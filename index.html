Getting started

Dependencies

This repository contains a set of native AngularJS extensions. The only required dependencies are:

  • AngularJS (requires AngularJS 1.2.x, tested with 1.2.10)
  • Bootstrap (tested with version 3.0.3). This version of the library (0.1.1) works only with Bootstrap in version 3.x.

Files to download

Build files for all extensions are distributed in several flavours: minified for production usage, un-minified for development. All the options are described and can be downloaded from here.

Installation

As soon as you've got all the files downloaded and included in your page you just need to declare a dependency on the ngOst module:

angular.module('myModule', ['ngOst']);

(filter) Blank Data in ngOst.filters.blankData

Value Argument Output
{{values.stringValue}} {{values.stringValue | blankData}}
null value "NULL VALUE" {{values.nullValue | blankData:"NULL VALUE"}}
null value {{values.nullValue | blankData}}
undefined value "N/A" {{values.undefinedValue | blankData:"N/A"}}
{{values.numericValue}} "NO DATA" {{values.numericValue | blankData:"NO DATA"}}
{{values.zero}} "NO DATA" {{values.zero | blankData:"NO DATA"}}

Description

Formats a value to display a placeholder when a value is null or undefined

Usage

In HTML Template Binding

{{ expression | blankData[:placeholder] }}

In JavaScript

$filter('blankData')(expression[, placeholder]);

Parameters

expression - value - input to filter

placeholder (optional) - string - placeholder value to be displayed.

Returns

string Formatted string.
