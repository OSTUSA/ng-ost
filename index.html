Getting started

Dependencies

This repository contains a set of native AngularJS extensions. The only required dependencies are:

  • AngularJS (requires AngularJS 1.2.x, tested with 1.2.10)
  • Bootstrap (tested with version 3.0.3). This version of the library (0.1.0) works only with Bootstrap in version 3.x.

Files to download

Build files for all extensions are distributed in several flavours: minified for production usage, un-minified for development. All the options are described and can be downloaded from here.

Installation

As soon as you've got all the files downloaded and included in your page you just need to declare a dependency on the ngost module:

angular.module('myModule', ['ngost']);

Blank Data (ngost.filters.blankData)

Value 1 Value 2
{{data.value1 | blankData}} {{data.value2 | blankData}}
{{data.value1 | blankData:"N/A"}} {{data.value2 | blankData:"N/A"}}

This is used to provide default values when a value is not present.

<table class="table table-bordered" ng-controller="blankDataDemoCtrl">
    <tr>
        <td>Value 1</td>
        <td class="text-right">Value 2</td>
    </tr>
    <tr>
        <td>{{data.value1 | blankData}}</td>
        <td class="text-right">{{data.value2 | blankData}}</td>
    </tr>
    <tr>
        <td>{{data.value1 | blankData:"N/A"}}</td>
        <td class="text-right">{{data.value2 | blankData:"N/A"}}</td>
    </tr>
</table>
 
function blankDataDemoCtrl($scope) {
    $scope.data = {
        'value1':'Value 1 String'
    };
}

Pretty Date (ngost.filters.prettyDate)

{{data.NewDate | prettyDate}}

Used to provide pretty dates to users based up a UTC Date Time

<table class="table table-bordered" ng-controller="prettyDateDemoCtrl">
    <tr>
        <td>{{data.NewDate | prettyDate}}</td>
    </tr>
</table>
 
function prettyDateDemoCtrl($scope) {
    $scope.data = {
        'NewDate':(new Date()).setDate(-2)
    };
}

String Format (ngost.filters.stringFormat)

{{data.message}}
{{data.message | stringFormat:'quick':'brown'}}
{{data.message | stringFormat:'slow':'green'}}

Used to provide a quick alternative to string format on the client side in tempaltes

<table class="table table-bordered" ng-controller="stringFormatDemoCtrl">
    <tr>
        <td>{{data.message}}</td>
    </tr>
    <tr>
        <td>{{data.message | stringFormat:'quick':'brown'}}</td>
    </tr>
    <tr>
        <td>{{data.message | stringFormat:'slow':'green'}}</td>
    </tr>
</table>
 
function stringFormatDemoCtrl($scope) {
    $scope.data = {
        'message': 'The {0} {1} fox jumped over the lazy log'
    };
}

Truncate (ngost.filters.truncate)

{{data.message}}
{{data.message | truncate}}

Used to truncate a string to a specified length

<table class="table table-bordered" ng-controller="truncateDemoCtrl">
    <tr>
        <td>{{data.message}}</td>
    </tr>
    <tr>
        <td>{{data.message | truncate}}</td>
    </tr>
</table>
 
function truncateDemoCtrl($scope) {
    $scope.data = {
        'message': 'The quick brown fox jumped over the lazy log'
    };
}

Us Phone Number (ngost.filters.usPhoneNumber)

{{data.value1}}
{{data.value1 | usPhoneNumber}}

Used to output a string or number into an acceptable Us Phone Number 1(555)123-4567

<table class="table table-bordered" ng-controller="usPhoneNumberDemoCtrl">
    <tr>
        <td>{{data.value1}}</td>
    </tr>
    <tr>
        <td>{{data.value1 | usPhoneNumber}}</td>
    </tr>
</table>
 
function usPhoneNumberDemoCtrl($scope) {
    $scope.data = {
        'value1': '5551234567'
    };
}
